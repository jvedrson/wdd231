Tools Included in Database

This project features 35 backend development tools including:

  • Frameworks: Express, Django REST Framework, Spring Boot, FastAPI, Flask, NestJS, Gin
  • Databases: Redis, MongoDB, PostgreSQL, MySQL, Apache Cassandra, Memcached
  • ORMs: GORM, Prisma, Sequelize, SQLAlchemy, Hibernate, TypeORM
  • DevOps: Docker, Kubernetes, Terraform, Jenkins, Git
  • Testing: Postman, Jest
  • Monitoring: Prometheus, Grafana
  • Messaging: Apache Kafka, RabbitMQ, Celery
  • API Tools: GraphQL, Swagger
  • Search: Elasticsearch
  • Servers: Nginx

Note: All tool information is publicly available and used for educational purposes. No proprietary or copyrighted content has been reproduced without permission.
