Submission Guidelines

What We're Looking For

  • Production-ready tools used in real applications
  • Active development and community support
  • Clear documentation and learning resources
  • Tools that solve common backend development problems

What We Don't Include

  • Deprecated or abandoned projects
  • Tools with limited documentation
  • Proprietary or closed-source tools without free tiers
  • Tools that are not specifically for backend development

Review Process

All submissions are reviewed by our team. We'll evaluate tools based on their utility, documentation quality, community support, and relevance to backend development. Approved tools will be added to our collection within 1-2 weeks.
